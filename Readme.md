# Face_Predict_Keras

(2022/9/24)

Qiita記事：[Kerasで顔識別](https://qiita.com/kiwsdiv/items/d5f6caf0f1a836b1bcf9)

# Requirement

- 適当なブラウザ
- Google アカウント
- Google Colaboratory
- Google Drive

# Usage
- Google Drive 学習用、テスト用の画像を用意。画像収集については別途記載。
- Google ColaboのランタイムのタイプをGPUに設定して実行する
